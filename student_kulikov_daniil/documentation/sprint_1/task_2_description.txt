**Зачем это было сделано? Чему научился в процессе выполнения задания?**
В первую очередь понять для чего билд системы нужны. Далее как выглядит
структура Gradle и какие преимущества она дает. Для себя я уяснил две вещи:
- Первое билд системы такие как Gradle отвечают за автоматизацию сборки
наших проектов
- Второе структура билд системы такой как Gradle, выглядит следующим
образом:
    1. Plugins — для подключения необходимых плагинов.
    2. dependencies — для указания зависимостей проекта.
    3. repositories — для определения репозиториев,
       откуда будут загружаться зависимости.
